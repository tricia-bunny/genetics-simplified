# Genetics: Oversimplified

A text-based fish breeding genetics simulation game in Python.

## Features
- Breed fish with inherited genetic traits (strength, intelligence, agility, etc.)
- Explore biomes on an ASCII map
- Complete adventures and collect items
- Genetic color inheritance system

## Usage
```bash
python genetics-oversimplified.py
```

## Files
- `genetics-oversimplified.py` - Main game
- `rpg.py` - RPG mechanics  
- `personalities.py` - Character systems
- `anagramatical.py` - Word puzzles
